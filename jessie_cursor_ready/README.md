# Jessie site – Cursor-ready project

- Główny plik strony: **index.html** w katalogu głównym.
- Dodatkowe zasoby (obrazy, fonty itp.) znajdują się w podkatalogach, np. `static.showit.co`, `fonts.gstatic.com`.
- Styl i skrypty są wczytywane z CDN (np. Google Fonts, jQuery, Animate.css). Jeśli chcesz, możesz dodać własne pliki `.css` i `.js` i podpiąć je w `<head>` lub przed `</body>` w `index.html`.

Projekt jest gotowy do otwarcia i edycji w Cursorze.
